# Quick Start

1. Setting up repo

```bash
git clone https://github.com/weiaiF/cpcc
```

2. Install Dependencies

```bash
conda create -n cpcc python=3.7
conda activate cpcc
cd cpcc
pip install -r requirements.txt
```
3. Train

```bash
sh train_cpcc.sh
```
